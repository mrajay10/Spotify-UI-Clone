# 🎵 Spotify UI Clone

This is based on HTML and CSS

A pixel-perfect clone of the Spotify Web Player built using pure **HTML** and **CSS**. This project replicates the core layout and design of Spotify's interface including the sidebar, music cards, and music player bar.

---

## 🖥️ Live Preview

> Open `Index.html` in your browser to view the project locally.

## ✨ Features

- 🎨 Spotify-inspired dark UI design
- 📚 Sidebar with **Home**, **Search**, and **Your Library** navigation
- 🃏 Music **cards** for Recently Played, Trending, and Featured Charts sections
- 🎵 Fixed **Music Player bar** at the bottom with:
  - Album art and song info
  - Playback controls (shuffle, previous, play/pause, next, repeat)
  - Progress/seek bar
  - Volume control
- 📱 Responsive — hides certain elements on screens below 1000px
